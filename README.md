# FlipMart Sales Analysis 

A complete sales analytics project built using Microsoft Excel and Power BI, based on the FlipMart Sales Dataset.
This project delivers insights on sales performance, product categories, customer segments, profitability, and shipping metrics.

The dataset includes detailed sales transactions with the following fields :

Order Info: Order ID, Order Date, Ship Date, Ship Mode

Customer Info: Customer ID, Customer Name, Segment

Geography: City, State, Country, Region, Market

Products: Category, Sub-Category, Product Name

Financials: Sales, Quantity, Profit, Shipping Cost

Others: Order Priority

Project Objectives :

Using Excel & Power BI, this project aims to :

Analyze total sales by category & sub-category

Measure profit contribution across markets & regions

Identify top-selling products

Track customer segment performance

Evaluate shipping cost & delivery time

Build an interactive Power BI dashboard
